# 🚀 TO Agent Browser

Smart AI Chat Agent running in your browser using your own API key.

---

## 🧠 About

TO Agent Browser is a lightweight web-based AI assistant that allows users to interact with advanced AI models directly from their browser.

Users can connect using their own API keys, making the tool flexible, private, and easy to use.

---

## ✨ Features

* 🔑 Use your own API key
* 💬 Real-time chat interface
* ⚡ Fast and lightweight
* 🌐 Runs entirely in the browser
* 🔒 No data stored (client-side usage)

## 🚀 Getting Started

1. Clone the repository:

```
git clone https://github.com/your-username/to-agent-browser.git
```

2. Open the project:

```
cd to-agent-browser
```

3. Run the project:

* Open `index.html` in your browser
